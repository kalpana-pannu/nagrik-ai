# NAGRIK AI

## About
IoT + AI platform for real-time civic monitoring
and citizen-government communication in India.

## Hardware
- ESP32 Development Board
- DHT22 Temperature and Humidity Sensor
- MQ135 Air Quality Sensor
- LDR Light Sensor
- Rain Sensor
- SW-420 Vibration Sensor
- Sound Sensor KY-038
- ML8511 UV Sensor
- LCD 16x2 I2C Display
- Active Buzzer
- LEDs and Resistors
- SOS Push Button
