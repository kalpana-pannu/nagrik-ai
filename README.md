# NAGRIK AI
### Civic Intelligence Platform
*"The Smartest Government Officer That Never Sleeps"*

## About
IoT + AI platform for real-time civic monitoring and citizen-government communication in India.

## Built By
**Kalpana Pannu** — First Year Engineering Student
Greater Noida, Uttar Pradesh
Started: June 2026

## Hardware
- ESP32 Development Board
- DHT22 Temperature and Humidity Sensor
- MQ-135 Air Quality Sensor
- LDR Light Sensor
- Rain Sensor
- SW-420 Vibration Sensor
- Sound Sensor KY-038
- ML8511 UV Sensor
- LCD 16x2 I2C Display
- Active Buzzer
- LEDs and Resistors
- SOS Push Button

## Current Status
Hardware build in progress. Working and tested: DHT22, LDR, SW-420 (vibration), MQ-135 (air quality), rain sensor, and buzzer. KY-038 sound sensor pending replacement (current unit suspected defective). ML8511 UV sensor, LCD display, and SOS button not yet integrated.
